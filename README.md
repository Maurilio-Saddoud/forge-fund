# Ship the Thing (POC)

Tagline: **ship the thing**

Proof-of-concept website for a scholarship initiative funding annual Claude Code or Codex Max plans.

## Included
- Sleek landing page (founder vibe)
- Donor mock flow (name, email, donation amount)
- Recipient mock application flow
- GitHub Pages deployment workflow

## Run locally

```bash
npm install
npm run dev
```

## Deploy
Push to `main`, then enable GitHub Pages in the repository settings (Source: GitHub Actions).
